Summary: Latitute vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this anlysis, we first pulled data from the OpenWeatherMap API to assemgle a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitute. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as as explanations and descriptions of any trends and correlations witnessed.
